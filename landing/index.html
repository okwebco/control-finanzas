Tecnología utilizada

Construido para durar

Tecnología moderna, probada y de bajo costo operativo.

  • FastAPI (Python 3.12) — interfaz de programación REST
  • PostgreSQL en Neon — base de datos sin servidor
  • Google Cloud Run — contenedores en la nube
  • JavaScript + HTML5 — sin marcos de trabajo externos
  • Cloudflare — distribución global y cifrado HTTPS
  • GitHub Actions — integración y despliegue continuos
  • Green API — notificaciones por WhatsApp
  • Gestor de credenciales GCP — claves fuera del código
  • Docker — empaquetado y registro de imágenes
  • pyotp — autenticación de dos factores
  • openpyxl — exportación a Excel
  • APScheduler — alertas automáticas diarias
